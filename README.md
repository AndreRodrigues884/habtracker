# HabTracker

Aplicação para acompanhamento de hábitos, permitindo que os usuários criem e acompanhem seus hábitos diários.

## Funcionalidades

- Registo e login de utilizadores com autenticação JWT.
- Criação, edição e exclusão de hábitos personalizados.
- Controle de frequência e progresso dos hábitos.
- Interface simples e intuitiva para gestão dos hábitos.

## Tecnologias

- Node.js
- Express
- MongoDB com Mongoose
- JWT para autenticação
- dotenv para variáveis de ambiente
